## Build Setup

```bash
# install dependencies
yarn install

# 開発環境を立ち上げる
yarn dev

# テストを実行
yarn test

# lintを実行
yarn lint

# lintの問題を解決
yarn lint:fix

# ライブラリアップデート
yarn upgrade-interactive
```
